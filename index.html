Welcome to TypeMVC

TypeMVC is a Web Framework that makes it easy to build enterprise-grade NodeJS apps using TypeScript.

Installation

Install the latest stable release of TypeMVC:

$ sudo npm install -g typemvc-cli

Note: Don’t use sudo if you’re using Windows

Create a New Project

Creating a new application is easy, TypeMVC comes with a number of generators that will help you creating everything necessary so you can start working on your project.

To create a new application:

$ tfw new testProject

Switch to your new app’s folder and run your app:

$ cd testProject
$ tfw start

And Voilà, your application has started. Visit http://localhost:3000 and you should see the default homepage.
